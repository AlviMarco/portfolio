
# Award-Winning Portfolio V2

## Features
- **Design:** Premium "Aurora" Dark theme, Glassmorphism, Responsive.
- **Animations:** GSAP ScrollTriggers, Text Scrambles, Hover effects.
- **Backend:** Supabase (Projects, Content).
- **Admin Panel:** Manage projects at `/admin`.

## Admin Access
- URL: `http://localhost:3000/admin`
- User: `admin`
- Pass: `admin123`

## Tech Stack
- Next.js 14 (App Router)
- Supabase (Postgres Database)
- GSAP (Animations)
- CSS Modules (Vanilla CSS per request)

## Setup
1. `npm install`
2. `npm run dev`
3. Enjoy!
